# WorkLogGT · 工作日志生成工具

此工具仅适用于**中文**用户, 因为它的界面和生成内容只有中文的.

## 文件结构
```
WorkLogGT-master/
├── gt-config.json 配置文件
├── index.js 窗体相关脚本
├── package-lock.json 依赖包小版本锁定配置
├── package.json 项目基础信息配置
├── page.html 窗体内置页面
├── script.js 生成相关脚本
```
